I have experience using VSC in the past to create websites and link them to Github, so by now I've learned to enjoy it. I probably enjoy the use of extensions the most and how easy it is to customize the editor as a whole to your own liking. The connection to Github is also extremely useful. Some of the autofill functions can get in the way when attempting to type large paragraphs such as these. The method of website previewing also requires an additional browser tab which is usually the case, but loses out to an editor such as Atom which offers the preview within the editor.
